Todo sobre Hosting, Dominio, Landing Page y Página Web

Conoce los conceptos esenciales para tener presencia profesional en Internet.

🌐 ¿Qué es un Dominio?

Un dominio es el nombre único que identifica a un sitio web en Internet. Es lo que escribes en la barra de direcciones de tu navegador para visitar una página, como www.tupagina.com.

Los dominios tienen extensiones como .com, .org, .edu, .pe, entre otros.

Ejemplo: www.zapatexpress.com

✅ Tipos de dominios:

  • Genéricos (gTLD): .com, .org, .net
  • Geográficos (ccTLD): .pe, .es, .mx
  • Patrocinados: .edu, .gov, .mil
  • Nuevos: .store, .tech, .blog

💾 ¿Qué es el Hosting?

El hosting permite almacenar los archivos de tu web para que esté disponible en Internet.

  • Compartido
  • VPS
  • Dedicado
  • En la nube

🌟 Hosting más utilizados:

  • Bluehost
  • HostGator
  • SiteGround
  • GoDaddy
  • Hostinger

🎯 ¿Qué es una Landing Page?

Una Landing Page es una página creada para una acción específica como:

  • Registrarse
  • Comprar un producto
  • Solicitar información

📄 ¿Qué es una Página Web?

Es un documento digital accesible por Internet. Suele contener:

  • Inicio
  • Acerca de
  • Servicios o productos
  • Contacto

Se crea con HTML, CSS, JS, y se aloja en un hosting con dominio.
